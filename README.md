# 🏦 Bank System (CLI Project)

## 📌 Description
This is a simple Command Line Interface (CLI) Bank Management System built using Python.

It allows users to create accounts, deposit money, withdraw money, transfer funds, and check balances.

The project demonstrates Object-Oriented Programming (OOP) and basic in-memory data handling using Python dictionaries.

---

## 🚀 Features
- Create bank account
- Deposit money
- Withdraw money
- Transfer money between accounts
- Check account balance
- Simple CLI menu system

---

## 🧠 Concepts Used
- Object-Oriented Programming (Classes & Objects)
- Functions
- Dictionaries (in-memory storage)
- Loops
- Conditional statements
- User input handling

##⚠️ Important Notes
-This project uses in-memory storage, so all data resets when the program stops
-Account names (or numbers in upgraded versions) are used to identify users
-No database or file storage is used in this version
